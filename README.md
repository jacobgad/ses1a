Before you start:

1. Install Node
2. Install MongoDB
3. Clone the repo
4. Set directory to local repo in CMD, E.G: 'cd C:\Users\lachl\Documents\Gitlab\ses6-restaurant'
5. run 'npm install'

To Start Server:

- Dev Mode: 'npm run dev'
- Prod Mode: 'npm start'

Open your browser to http://localhost:3000/
